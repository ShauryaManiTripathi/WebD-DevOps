Interactive CSS Notes - Zerodha Landing Page Example

These notes demonstrate how to build a basic structure of the Zerodha landing page using HTML and inline CSS styles, based on the CSS properties and Flexbox concepts we've learned.

2.8 Zerodha Landing Page Example

Explanation of CSS used in Zerodha Example:
  • Top Navigation Bar (First
    ):
    • display: flex; justify-content: space-between;: Uses Flexbox to position logo on the left and navigation links on the right, with space in between.
    • padding: 20px;: Adds padding around the navigation bar content.
    • box-shadow: 0 2px 5px #ddd;: Adds a subtle box shadow at the bottom of the navigation bar.
  • Logo and Navigation Links (Inside Top Bar):
    • Logo (): width: 120px; sets the logo width.
    • Navigation Links (
      and ): display: flex; on the container div arranges links horizontally. padding: 20px; color: #666; cursor: pointer; styles each link with padding, grey color, and pointer cursor on hover.
  • Main Landing Image (Second
    ):
    • display: flex; justify-content: center;: Centers the image horizontally using Flexbox.
    • padding-top: 60px;: Adds space above the image.
    • width: 900px;: Sets the image width.
  • Headings ("Invest in everything" and Subheading) (Third and Fourth
    ):
    • display: flex; justify-content: center;: Centers the headings horizontally.
